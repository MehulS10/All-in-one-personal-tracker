# Omnitrack is a personal workspace app I vibe-coded for myself to manage job applications, freelance leads, projects, and daily work in one place.

An offline-first personal workspace for:

- Job tracking
- Leads CRM
- Notes
- Project management
- Daily workflow organization

Built with Electron + React + Vite.

---

# Features

- Job application pipeline
- Leads CRM
- Workspace notes
- Project tracking
- Offline database support
- Global activity feed
- Fast desktop UI

# Installation & Running Locally

windows application 

## Prerequisites

Make sure you have installed:

- Node.js
- npm

Download Node.js here:

https://nodejs.org/

---

## Clone Repository

```bash
git clone https://github.com/MehulS10/All-in-one-personal-tracker.git
```

---

## Install Dependencies

```bash
npm install
```

---

## Run Application

```bash
npm run dev
```

This will start:
- Vite development server
- Electron desktop application

---

# Build Application

```bash
npm run build
```
